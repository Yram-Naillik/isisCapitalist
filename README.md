## Push du code

Voir les éléments modifés
```
git status
```
Ajouter les fichiers que l'on souhaite 
```
git add [chemin du fichier à ajouter]
```
Commit les fichiers modifiés ajoutés
```
git commit -m "[message]"
```
Pousser le code sur GitHub
```
git push
```
